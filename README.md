# 🌍 REST Countries Explorer

A modern and responsive **REST Countries Explorer** built using HTML, CSS, and JavaScript.
The application provides detailed information about countries worldwide with dynamic filtering, search functionality, and theme customization.

🔗 **Live Demo:** https://country-info-web-app.netlify.app/


## 📌 Overview

This application fetches data from a public API and displays all countries in a clean, responsive card layout.
Users can easily search for countries, filter them by region, and explore detailed information including neighboring (border) countries.

The app also includes a **Dark Mode / Light Mode toggle** to enhance user experience and accessibility.

---

## ✨ Core Features

* 🌐 Display all countries in a responsive card layout
* 🔍 Real-time search by country name
* 🌎 Filter countries by region (Asia, Europe, Africa, etc.)
* 🧾 Detailed country view on card click
* 🔗 Border countries navigation
* 🌗 Dark Mode / Light Mode toggle
* ⚡ Fast API-based dynamic data loading


## 🛠️ Technology Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla)**
* **REST Countries API**


## ⚙️ Project Workflow

1. Load all countries from API
2. Search or filter countries by region
3. Click on a country card
4. View detailed information
5. Explore border countries

REST Countries Explorer

A modern and responsive REST Countries Explorer built using HTML, CSS, and JavaScript.
The application provides detailed information about countries worldwide with dynamic filtering, search functionality, and theme customization.

🔗 Live Demo: https://country-info-web-app.netlify.app/

📌 Overview

This application fetches data from a public API and displays all countries in a clean card layout.
Users can search for countries, filter them by region, and explore detailed information including border countries.

The application also includes a Dark Mode / Light Mode toggle for a better user experience.

✨ Core Features
🌐 Display all countries in responsive card layout
🔍 Real-time search by country name
🌎 Filter countries by region (Asia, Europe, Africa, etc.)
🧾 Detailed country view on card click
🔗 Border countries navigation
🌗 Dark Mode / Light Mode toggle
⚡ Fast API-based dynamic data loading

🛠️ Technology Stack
HTML5
CSS3
JavaScript (Vanilla)
REST Countries API

⚙️ Project Workflow
Load all countries from API
Search or filter countries by region
Click on a country card
View detailed information
Explore border countries
